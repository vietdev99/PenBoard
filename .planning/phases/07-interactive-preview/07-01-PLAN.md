    Create `src/services/preview/preview-html-generator.ts`:

    This is the core preview engine. It builds self-contained HTML documents from PenDocument data. Does NOT import from html-generator.ts directly (to keep code-panel generator clean), but replicates the CSS generation patterns (fillToCSS, strokeToCSS, etc.) with preview-specific extensions.

    1. Internal CSS helper functions (replicate from html-generator.ts pattern):
       - `varOrLiteral(value: string): string` — if `isVariableRef(value)` return `var(${variableNameToCSS(value.slice(1))})`, else return value
       - `fillToCSS(fills: PenFill[]): Record<string, string>` — solid, linear_gradient, radial_gradient
       - `strokeToCSS(stroke: PenStroke): Record<string, string>` — border-width, border-style, border-color
       - `effectsToCSS(effects: PenEffect[]): Record<string, string>` — box-shadow
       - `cornerRadiusToCSS(cr): Record<string, string>` — border-radius
       - `layoutToCSS(node: ContainerProps): Record<string, string>` — flex layout, gap, padding, justify, align
       - `escapeHTML(text: string): string` — &, <, >, "

    2. `generateNodeHTML(node: PenNode, depth: number, rules: CSSRule[], connections: ScreenConnection[]): string`:
       - Uses `getSemanticTag(node.role, node.type, ...)` for tag selection instead of always `<div>`
       - Adds `data-nav-click="{sourceElementId}"` attribute on elements that have a ScreenConnection with matching sourceElementId and triggerEvent='click'
       - Adds `data-nav-hover="{sourceElementId}"` for triggerEvent='hover'
       - Adds `data-nav-submit="{sourceElementId}"` for triggerEvent='submit'
       - Adds `tabindex="0"` on elements with any connection (keyboard accessibility)
       - Adds `id="node-{node.id}"` on every element (for targetFrameId scrolling)
       - For `type: 'text'`: use semantic tag based on role or fontSize heuristic
       - For `type: 'image'`: `<img src="{node.src}" alt="{name}" />` — src preserved as-is (already base64/data URL from document)
       - For `type: 'path'` with iconId: inline SVG `<svg viewBox="0 0 w h"><path d="{node.d}" fill="{color}" /></svg>`
       - For `type: 'ref'`: should NOT reach here (already expanded by resolver), but fallback to `<!-- Component: {ref} -->`
       - For role 'input'/'form-input': `<input type="text" placeholder="{content}" />`
       - For role 'textarea': `<textarea placeholder="{content}"></textarea>`
       - For role 'checkbox': `<input type="checkbox" />`
       - For role 'radio': `<input type="radio" />`
       - For data-bound 'dropdown'/'select' role: `<select>` with `<option>` per entity row

    3. Export `generatePageHTML(pageChildren: PenNode[], connections: ScreenConnection[]): { html: string; css: string }`:
       - Reset class counter, generate all nodes, collect CSS rules
       - Returns `{ html: '<div class="page-root">...nodes...</div>', css: rulesString }`

    4. Export `generatePreviewHTML(doc: PenDocument, pageId: string | null, frameId: string | null, previewId: string): string`:
       - Determine target page: find page by pageId, fallback to first screen page
       - Determine target nodes: if frameId, find frame in page children and use its children; else use page children
       - Call `resolvePageForPreview(targetNodes, doc)` to resolve refs/data/variables
       - Call `generatePageHTML(resolvedNodes, doc.connections ?? [])` to produce HTML+CSS
       - Generate CSS variables block via `generateCSSVariables(doc)` from css-variables-generator
       - Build full HTML document string:
         ```
         <!DOCTYPE html>
         <html lang="en">
         <head>
           <meta charset="utf-8" />
           <meta name="viewport" content="width=device-width, initial-scale=1" />
           <meta http-equiv="Content-Security-Policy" content="default-src 'none'; script-src 'unsafe-inline'; style-src 'unsafe-inline'; img-src data: blob:; font-src data:;" />
           <title>Preview - {pageName}</title>
           <style>
             * { margin: 0; padding: 0; box-sizing: border-box; }
             body { overflow: auto; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; }
             .page-container { display: none; position: relative; }
             .page-container.active { display: block; }
             {cssVariables}
             {generatedCSS}
           </style>
         </head>
         <body>
           <div id="page-{pageId}" class="page-container active">
             {generatedHTML}
           </div>
           <script>
             // Placeholder for navigation JS (added by Plan 03)
             // SSE hot reload listener
             (function() {
               var evtSource = new EventSource('/api/preview/events?id=' + '{previewId}');
               evtSource.onmessage = function(e) {
                 var data = JSON.parse(e.data);
                 if (data.type === 'preview:update') window.location.reload();
               };
             })();
           </script>
         </body>
         </html>
         ```
       - Return the full HTML string
