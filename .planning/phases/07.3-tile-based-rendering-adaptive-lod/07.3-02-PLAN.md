# Plan 07.3-02: FPS-Based Adaptive LOD System

## Overview
Replace the static `benchmarkPending` mechanism with a real-time FPS monitor
that auto-adjusts render tier. Integrates with existing LOD from Phase 07.2.

**Wave:** 1 (parallel with Plan 01)
**Decisions:** D3 (FPS-based adaptive)

## Tasks

### Task 1: FPSMonitor class
**File:** `src/canvas/skia/fps-monitor.ts` [NEW]

```typescript
type RenderTier = 'full' | 'quick' | 'tile'

class FPSMonitor {
  private frameTimes: number[] = []  // rolling window of last 10 frames
  private _tier: RenderTier = 'full'
  private consecutiveSlow = 0  // frames > 16ms
  private consecutiveFast = 0  // frames < 10ms

  get tier(): RenderTier

  recordFrame(elapsedMs: number): void {
    // Push elapsed, keep last 10
    // If > 16ms for 3 consecutive → decrease tier
    // If < 10ms for 5 consecutive → increase tier
    // Tier order: full → quick → tile
  }

  reset(): void  // reset to 'full' (e.g. on page change)
}
```

**UAT:**
- 3 frames at 20ms → tier drops from 'full' to 'quick'
- 5 frames at 8ms → tier increases from 'quick' to 'full'
- Already at 'tile' + slow frames → stays at 'tile'
