## Tasks

### Task 1: Zoom level transition with scaled tiles
**File:** `src/canvas/skia/skia-engine.ts`

When zoom crosses a resolution threshold (e.g., 25% → 26%):
1. Keep old zoom level tiles cached
2. Draw old tiles scaled to new zoom (blurry but instant)
3. Mark all tiles at new zoom level as dirty
4. Progressive render replaces blurry tiles with sharp ones

```typescript
// In render():
// Detect zoom level change
const newTileSize = this.tileManager.getTileSize(this.zoom)
if (newTileSize !== this.lastTileSize) {
  // Zoom level crossed — scale old tiles temporarily
  this.scaledTileZoom = this.lastZoom
  this.lastTileSize = newTileSize
  this.tileManager.markAllDirty()
}

// If scaledTileZoom set and tile is dirty:
// Try to blit tile from previous zoom level (scaled)
// Once re-rendered at correct zoom → clear scaledTileZoom for that tile
```

**UAT:**
- Zoom from 20% to 30%: tiles appear instantly (scaled from old)
- Sharp tiles progressively replace blurry ones
- No visual glitch during transition

### Task 2: Delta dirty-marking during drag
**File:** `src/canvas/skia/skia-engine.ts`, `src/canvas/skia/skia-canvas.tsx`

During node drag (position change only, not full sync):
- Calculate moved node's old bounding box → mark old tiles dirty
- Calculate new bounding box → mark new tiles dirty
- Only ~2-4 tiles re-render instead of full canvas

```typescript
// In drag handler (skia-canvas.tsx), after updating node position:
engine.tileManager.markDirty({ x: oldX, y: oldY, w: nodeW, h: nodeH })
engine.tileManager.markDirty({ x: newX, y: newY, w: nodeW, h: nodeH })
engine.markDirty()
```

**UAT:**
- Drag node → only tiles under old+new position re-render
- Rest of canvas stays cached (no re-render)
- Smooth drag at any zoom level

### Task 3: Cleanup and dispose
**File:** `src/canvas/skia/skia-engine.ts`

Ensure proper cleanup:
- `dispose()` → `tileManager.dispose()` (delete all SkImages)
- `resize()` → `tileManager.markAllDirty()` + recreate tile surface
- Page change → `tileManager.dispose()` + rebuild

**UAT:**
- No GPU memory leaks after page switch
- Resize triggers full tile refresh
- DevTools Memory tab shows stable GPU memory
