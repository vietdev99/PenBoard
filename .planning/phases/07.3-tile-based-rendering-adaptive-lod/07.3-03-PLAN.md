### Task 2: Refactor render() — tile blitting pipeline
**File:** `src/canvas/skia/skia-engine.ts`

Replace current render loop with tile pipeline:

```typescript
render() {
  // ... existing clear + viewport transform ...

  const tileSize = this.tileManager.getTileSize(this.zoom)
  const visibleKeys = this.tileManager.getVisibleTileKeys(
    vpLeft, vpTop, vpRight, vpBottom, this.zoom
  )

  // Phase A: Blit all clean tiles (cached SkImages)
  for (const key of visibleKeys) {
    const tile = this.tileManager.getOrCreate(key)
    this.tileManager.touch(key)
    if (!tile.dirty && tile.image) {
      // Blit cached image at tile position
      canvas.drawImage(tile.image, tile.sceneRect.x, tile.sceneRect.y)
    }
  }

  // Phase B: Render dirty tiles (budgeted — max 5 per frame)
  const dirtyKeys = visibleKeys.filter(k => {
    const t = this.tileManager.tiles.get(k)
    return t && (t.dirty || !t.image)
  })

  const budget = Math.min(5, dirtyKeys.length)
  for (let i = 0; i < budget; i++) {
    const key = dirtyKeys[i]
    const nodes = this.nodeMap.get(key) ?? []
    this.tileManager.renderTile(ck, tile, nodes, this.renderer,
                                 selectedIds, this.fpsMonitor.tier)
    // Blit freshly rendered tile
    canvas.drawImage(tile.image, tile.sceneRect.x, tile.sceneRect.y)
  }

  // Schedule next frame if dirty tiles remain
  if (dirtyKeys.length > budget) this.markDirty()

  // Phase C: Overlays (selection, hover, connections, labels, agent)
  // ... existing overlay code unchanged ...
}
```

**UAT:**
- Clean tiles blit from cache (zero render cost)
- Dirty tiles render progressively (max 5/frame)
- Remaining dirty tiles trigger next frame
- Overlays render on top of tiles
