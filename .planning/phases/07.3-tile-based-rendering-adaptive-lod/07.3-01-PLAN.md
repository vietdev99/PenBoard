# Plan 07.3-01: TileManager Core + Multi-Resolution Grid

## Overview
Build the core `TileManager` class that manages the tile grid, LRU cache, and dirty tracking.
This is the foundation all other plans depend on.

**Wave:** 1 (no dependencies)
**Decisions:** D1 (full tile grid), D2 (dirty flags), D5 (multi-res), D6 (LRU cache)

## Tasks

### Task 1: TileManager class + tile grid math
**File:** `src/canvas/skia/tile-manager.ts` [NEW]

Create `TileManager` class:

```typescript
interface TileEntry {
  key: string           // "z2_x3_y5" (zoomLevel_col_row)
  image: Image | null   // cached SkImage
  dirty: boolean
  lastUsed: number      // performance.now()
  sceneRect: { x: number, y: number, w: number, h: number }
}

class TileManager {
  private tiles = new Map<string, TileEntry>()
  private maxTiles = 150  // LRU cache limit

  // Multi-resolution zoom levels (Decision 5)
  private zoomLevels = [
    { maxZoom: 0.05,  tileSize: 2048 },  // overview
    { maxZoom: 0.25,  tileSize: 1024 },
    { maxZoom: 1.0,   tileSize: 512 },
    { maxZoom: Infinity, tileSize: 256 }, // detail
  ]

  getTileSize(zoom: number): number
  getVisibleTileKeys(vpLeft, vpTop, vpRight, vpBottom, zoom): string[]
  getTileSceneRect(key: string): { x, y, w, h }
  markDirty(sceneRect: { x, y, w, h }): void  // mark tiles intersecting rect
  markAllDirty(): void
  getOrCreate(key: string): TileEntry
  touch(key: string): void    // update lastUsed
  evictLRU(): void            // evict if > maxTiles
  dispose(): void             // delete all SkImages
}
```

**UAT:**
- `getTileSize(0.03)` returns 2048
- `getTileSize(0.5)` returns 512
- `getVisibleTileKeys()` returns correct keys for viewport
- `markDirty(rect)` marks overlapping tiles

### Task 2: LRU eviction + SkImage lifecycle
**File:** `src/canvas/skia/tile-manager.ts`

Implement proper SkImage lifecycle:
- `evictLRU()` — when `tiles.size > maxTiles`, delete oldest `lastUsed` entries
- Call `image.delete()` on eviction (prevent GPU memory leak)
- `dispose()` — delete all cached SkImages on engine shutdown

**UAT:**
- Cache grows to 150, evicts oldest on 151st
- `dispose()` calls `delete()` on all cached images

### Task 3: Node-to-tile mapping
**File:** `src/canvas/skia/tile-manager.ts`

Add method to assign renderNodes to tiles:
```typescript
buildNodeMap(renderNodes: RenderNode[], zoom: number): Map<string, RenderNode[]>
```
- For each node, find all tiles its bounding box intersects
- Large nodes (spans >4 tiles) → add to each tile
- Result: per-tile node lists for rendering

**UAT:**
- Node at (0,0,100,100) with tileSize 256 → only in tile "z2_x0_y0"
- Node at (200,200,300,300) spanning tile boundary → in multiple tiles
