# Phase 09.1 — PenBoard Workspace & External Mermaid Docs

## Context

Mermaid business flows hiện lưu trong page context (bên trong .pb file), gây phình file và không tách biệt. Cần kiến trúc workspace riêng để AI có thể đọc hiểu toàn bộ dự án (UI design + business flows) khi nhận file nén.

## Decisions

### A. Workspace Directory Structure

| Decision | Choice |
|----------|--------|
| Workspace dir | `.penboard/` tại project root, bên cạnh file `.pb` |
| Flow storage | `.penboard/flows/*.md` — mỗi flow 1 file markdown chứa mermaid + description |
| Context docs | `.penboard/docs/*.md` — AI-generated context documents |
| Manifest | `.penboard/manifest.json` — index of all docs |
| Workspace link | `.pb` file lưu `workspace` field (relative path tới project root chứa `.penboard/`) |
| Zip package | `.pb` + `.penboard/` = complete project, AI đọc hiểu hoàn toàn |

### B. UI Layout — Canvas / Flow Tabs

| Decision | Choice |
|----------|--------|
| Top-level tabs | 2 tab chính: **Canvas** / **Flow**, đặt **trên toolbar** |
| Flow tab content | Single scrollable page tổng hợp tất cả `.penboard/flows/*.md` |
| Flow navigation | TOC (Table of Contents) ở đầu trang với jump links tới từng section |
| Each section | Header + rendered Mermaid SVG + description text từ .md |
| Flow editing | **View-only** trong UI — AI sửa qua MCP tools |
| Canvas tab | Giữ nguyên editor hiện tại với page tabs (modules) bên dưới |

### C. MCP Tools

| Tool | Purpose |
|------|---------|
| `write_flow(name, content)` | Ghi/cập nhật mermaid flow vào `.penboard/flows/{name}.md` |
| `read_flow(name)` | Đọc 1 flow file |
| `list_flows()` | Liệt kê tất cả flow files |
| `write_doc(category, name, content)` | Ghi tài liệu context vào `.penboard/docs/` |
| `read_doc(category, name)` | Đọc tài liệu context |

### D. Connection Arrows Bug Fix

| Decision | Choice |
|----------|--------|
| Bug | Khi drag node có connection, mũi tên không di chuyển theo |
| Root cause | `rnMap` không được update trong drag loop |
| Fix | Đã code xong — sync `rnMap` trong drag handler |
| Commit | Commit riêng trước phase, không bundle vào |

## File Format: `.penboard/flows/*.md`

```markdown
# Authentication Flow

Mô tả luồng xác thực người dùng từ login đến dashboard.

```mermaid
flowchart TD
    Start([User Opens App]) --> HasAccount{Has Account?}
    HasAccount -->|Yes| LoginScreen[Login Screen]
    ...
`` `

## Notes

- Luồng bao gồm cả error handling và session management
- Liên kết với screens: Login Screen, Register Screen, Dashboard
```

## Out of Scope

- Edit mermaid trực tiếp trong UI (chỉ view-only, AI sửa qua MCP)
- Mermaid render engine (dùng library có sẵn: mermaid.js)
- Export flow to image (có thể thêm sau)
