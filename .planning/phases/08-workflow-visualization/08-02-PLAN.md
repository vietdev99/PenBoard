    Hook implementation:
    1. Lazy-load mermaid: `const mermaidRef = useRef<typeof import('mermaid').default | null>(null)`. On first call to `regenerate()`, if null, do `const m = await import('mermaid'); m.default.initialize({ startOnLoad: false, securityLevel: 'loose', theme: 'base', flowchart: { curve: 'basis', padding: 20, nodeSpacing: 60, rankSpacing: 80 } }); mermaidRef.current = m.default`.
    2. State: `svgContent: string` (SVG HTML string), `graph: WorkflowGraph | null`, `isEmpty: boolean`, `isLoading: boolean`
    3. `regenerate()` async function:
       - Get `doc` from `useDocumentStore.getState().document`
       - Get `activePageId` and `workflowFocusMode` from `useCanvasStore.getState()`
       - Call `buildWorkflowGraph(doc)` -> fullGraph
       - If fullGraph.nodes.length === 0, set `isEmpty: true`, `svgContent: ''`, return
       - If workflowFocusMode and activePageId, call `filterToFocusGraph(fullGraph, activePageId)` -> filtered, else use fullGraph
       - Call `generateMermaidSyntax(filtered, activePageId)` -> syntax
       - Lazy-load mermaid if needed
       - Call `const { svg, bindFunctions } = await mermaidRef.current.render(\`wf-${++renderIdRef.current}\`, syntax)`
       - Set `svgContent = svg`, `graph = filtered`, `isEmpty = false`
       - Store `bindFunctions` in a ref for the panel to call after DOM insertion
    4. Subscribe to `useDocumentStore` using `createDebouncedUpdater` from Plan 01 (WF-03 auto-update):
       ```typescript
       useEffect(() => {
         const updater = createDebouncedUpdater(() => regenerate(), 500)
         const unsub = useDocumentStore.subscribe(() => updater.trigger())
         return () => { unsub(); updater.cleanup() }
       }, [])
       ```
    5. Also regenerate when `activePageId` or `workflowFocusMode` changes:
       ```typescript
       useEffect(() => { regenerate() }, [activePageId, workflowFocusMode])
       ```
       Read `activePageId` from `useCanvasStore((s) => s.activePageId)` and `workflowFocusMode` from `useCanvasStore((s) => s.workflowFocusMode)`.
    6. Register global click handler: `(window as any).__wfNavigate = (mermaidNodeId: string) => { ... }` that uses `buildMermaidIdMap` to resolve back to original ID, then calls `useCanvasStore.getState().setActivePageId(originalId)` for pages, or navigates to ERD page for entities. Clean up on unmount.
    7. Return: `{ svgContent, graph, isEmpty, isLoading, regenerate, bindFunctions: bindFunctionsRef }`
